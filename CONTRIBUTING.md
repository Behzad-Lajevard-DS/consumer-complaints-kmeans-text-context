# Contributing

Contributions are welcome when they preserve the methodological purpose of the project.

Useful contribution areas include:

- stability analysis across samples and random seeds;
- evaluation of K values beyond ten;
- near-duplicate and legal-template detection;
- sentence-level analysis;
- soft clustering or topic modelling;
- sentence-transformer embeddings;
- reproducibility and documentation improvements.

Please do not frame a higher metric alone as proof of a better scientific solution. Changes should explain how the method fits the corpus and the analytical objective.
